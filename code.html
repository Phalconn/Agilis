The Code for AGILIS


//get the values from the page
//starts or controller function
function getValues(){
    //get values from the page
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;
    
    //We need to validate our input
    //parse into Integers
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    if(Number.isInteger(startValue) && Number.isInteger(endValue)) {
        //we call generate Numbers
        let numbers = generateNumbers(startValue, endValue);
        //we call display Numbers
        displayNumbers(numbers);
    } else {
        alert("Error: Program is expecting an integer");
    }

    
}

//generate numbers from startvalue to endValue
//logic function(s)
function generateNumbers(sValue,eValue){

    let numbers = [];

    //we want to get all numbers from start to end
    for (let index = sValue; index <= eValue; index++) {
        //this will execute in a loop until index =eValue
        numbers.push(index);
    }
    return numbers;
}

//display numbers and mark even numbers bold
//display or view functions
function displayNumbers(numbers){

    let templateRows = "";

    for (let index = 0; index < numbers.length; index++) {
        
        let className = "even";
        let number = numbers[index];

        if(number % 2 == 0) {
            className = "even"
        }
        else{
            className = "odd"
        }
        templateRows += `${number}`;
    }
    //This does not render correctly with Prism see the source
    document.getElementById("results").innerHTML = templateRows;

}
}

The code is structured with several Javascript functions.

Getting Values

This is the first function. First the user is prompted to enter a start value and an end value. We created two varuables, namely startValue and endValue which will be retrived by using the the document method getElementById() and returns an element object representing the element whose id property matches the specified primitive type.

Validating Input

It is crucial that the program validates the type of input the user enters. In this program the user is expected to enter an integer. Therefore we validate the input by using a specific function called parseInt which is function that usually parses a string argument and returns an integer of the specified radix. However in this case in i expected to parse an integer by using the parameters startValue and endValue stored from the previous values.

We use an boolean to validate the use input which helps to itterate through the program and see if the input parameters previously set have been matched. If the target value is an integer, return true, otherwise return false. If the value is NaN or Infinity, return false. The method will also return true for floating point numbers that can be represented as integer.

If it fails to return true, the program will alert the user with a message showing that must enter an integer.

The let statement declares a block-scoped local variable, optionally initializing it to a value. In this case we call the numbers generated and asign it the variable generateNumbers.

Next is a for loop which will execute untill index is equal to end value and will return numbers.

Display numbers and mark bold

We have a funtion of displayNumbers which passes a parameter of numbers in a form of an array. There is a for loop to validate if the number is even or odd. two classes of "odd" and "even"

We then add a templateRows variable to store a string. This is a variable we are going to use to hold out HTML. We also have a for loop to itterate through our numbers starting from index 0 to the end of the array length and obtain the max number. In the for loop we used a let statement to declare the numbers variable to be index and also trmplate row to create a table with rows same number of times as in the array. For example if there are a hundred numbers in the numbers.length, then the templateRows would create a 100 rows and display them on the page.
